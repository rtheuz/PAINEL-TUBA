# PAINEL TUBA

Painel administrativo desenvolvido com **Google Apps Script** para centralizar operações, automações e acompanhamento de dados do projeto TUBA.

---

## 📌 Visão Geral

O **PAINEL TUBA** tem como objetivo facilitar a gestão de processos por meio de uma interface simples, conectada aos serviços do Google (como Planilhas, Drive, Gmail e Formulários), permitindo:

- automações de tarefas repetitivas;
- controle e visualização de dados;
- execução de rotinas administrativas;
- redução de trabalho manual.

---

## 🚀 Funcionalidades

- Interface de painel para ações rápidas;
- Integração com Google Sheets;
- Execução de funções Apps Script pelo painel;
- Organização de fluxos internos do projeto;
- Estrutura pronta para expansão de módulos.

---

## 🛠️ Tecnologias Utilizadas

- **Google Apps Script** (JavaScript)
- **Google Sheets** (base de dados / interface)
- **HTML/CSS/JavaScript** (quando aplicável em páginas web do Apps Script)

---

## 🔐 Permissões

Este projeto pode solicitar permissões para:

- ler/editar planilhas;
- acessar/gerenciar arquivos no Google Drive;
- executar ações em nome do usuário.

Conceda apenas em ambientes confiáveis.

---

## ✅ Boas Práticas

- Manter funções pequenas e reutilizáveis;
- Registrar erros com `Logger.log` ou tratamento com `try/catch`;
- Separar regras de negócio em arquivos `.gs` específicos;
- Documentar mudanças importantes.
